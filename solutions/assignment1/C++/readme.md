# MySQL C++ Example

## Prerequisites
Make sure you have the MySQL client library installed:

- **Ubuntu/Debian**
  ```bash
  sudo apt-get update
  sudo apt-get install libmysqlclient-dev mysql-client


# MySQL C++ Example

## Compile
```bash
g++ main.cpp -o mysql_test -lmysqlclient
```
## Run
```bash
./mysql_test
```
